## Design Context

### Users
ISU(이수시스템) 임직원이 업무 시간에 사무실 모니터 앞에서 사용하는 사내 업무 포털.
프로젝트 관리, 사내 위키(지식 베이스), 근태, 공지사항, AI 검색 등 일상 업무 도구.

### Brand Personality
**Structured, Trustworthy, Precise** — IT 시스템 전문기업의 내부 도구답게.
차분하고 신뢰감 있으면서도, 라임 그린 액센트로 활력과 현대적 감각을 부여.

### Aesthetic Direction
- **Theme**: Light mode (사무실 주간 사용 환경)
- **Main color**: ISU Blue `#003B94` — `oklch(0.38 0.157 260)` — 깊고 자신감 있는 코발트 블루
- **Secondary color**: ISU Lime `#7AC143` — `oklch(0.74 0.174 134)` — 선명한 연두 액센트
- **Neutrals**: 블루(hue 260) 방향으로 틴팅된 뉴트럴 스케일
- **Sidebar**: ISU 브랜드 딥 블루 배경 + 라임 액티브 상태
- **60-30-10**: 60% 블루-틴팅 뉴트럴 표면 / 30% ISU 블루 / 10% 라임 그린
- **Anti-references**: Tailwind 기본 gray+blue, Inter 폰트, 동일 카드 그리드, 좌측 스트라이프

### Design Principles
1. **Brand-first**: 모든 색상이 ISU 브랜드 블루/라임에서 파생. Tailwind 기본 팔레트 사용 금지.
2. **OKLCH native**: 지각적 균일성을 위해 OKLCH 색상 함수 사용. HSL/hex 사용 최소화.
3. **Typography hierarchy**: Lexend(display) + Noto Sans KR(body) 2폰트 시스템. 크기 비율 1.25 이상.
4. **No AI slop**: 좌측 스트라이프, gradient text, 동일 카드 그리드, rounded-2xl + shadow-sm 패턴 금지.
5. **Intentional rhythm**: 간격에 계층적 변화를 줌. 동일 padding 반복 지양.
