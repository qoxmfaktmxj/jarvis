# `.local/` — 로컬 전용 자료 보관소

이 폴더는 **개발 중 참조용 자료**(엑셀, PDF, 샘플 SQL, 스크래치 메모, 1회성 데이터 등)를 두는 곳입니다.

## 규칙

- 이 폴더 안의 모든 파일은 **git에 올라가지 않습니다** (`.gitignore`로 차단).
- 단 `.gitkeep`과 이 `README.md`만 예외적으로 추적되어, 폴더 자체는 클론 시 같이 옵니다.
- AI 어시스턴트(Claude/Codex 등)는 이 폴더 안 파일을 **읽을 수 있습니다** — 작업 컨텍스트 제공용.
- 민감 정보(비밀번호, 키, 고객 PII 등)는 절대 두지 마세요. AI가 읽고 응답에 포함될 수 있습니다.

## 사용 예시

```
.local/
├── 추가개발관리.xlsx          # 분석 의뢰용 엑셀
├── 취업규칙.pdf              # 정책 참조용 PDF
├── sample-cases/             # 샘플 케이스 데이터
│   └── case-001.json
├── scratch.md                # 스크래치 메모
└── tsmt001-fragment.sql      # 마이그레이션 일부 샘플
```

## "왜 git에 올리지 않나"

- 사내 데이터 / 고객 데이터는 외부 GitHub에 노출되면 안 됨.
- 개발자마다 다른 자료를 참조 — 공유 필요 없음.
- 1회성 마이그레이션 산출물은 운영 코드와 분리되는 게 명확.

운영에 영구히 필요한 데이터는 `data/` 또는 적절한 `packages/<도메인>/fixtures/`에 두세요.
