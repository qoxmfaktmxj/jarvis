# 2025년부터 변경된 다양한 HR제도

Company Home

**📢 HR제도 개편을 왜 하나요?**



HR제도 개편에 앞서 내부 직원 인터뷰를 진행한 결과 아래와 같은 피드백이 있었습니다.

아래 내용을 바탕으로 직급체계/보상제도에 대하여 HR제도 개편을 진행하였습니다.



- **직급체계** : 현재의 역할을 반영하지 못함
                    외부 프로젝트 진행 시 고객이 주니어급으로 인식하여 업무 진행에 제약이 발생

- **보상제도** : 동기부여 장치가 약함
                    연장근무에 대한 보상이 필요
                    PM에 대한 보상 필요

- **조직** : 부서간 R&R 조정 필요
           본부내 인력 순환 필요

- **육성제도** : 직무별 교육체계가 필요
                    필요한 교육을 개인이 찾아야 함



또한 육성제도를 포함한 교육/직무역량 체계와 관련해서는 별도의 프로젝트를 통하여 개선을 진행하고자 합니다.





**📢 직급체계는 어떻게 바뀌나요?**



2025년 이전까지는 사원/대리/과장/차장/부장 5단계 직급체계이었지만

2025년부터 선임/책임/수석 3단계 직급체계로 개편됩니다.



아래의 직급체계 링크를 통해 상세한 내용을 확인하세요.





**📢 보상체계는 어떻게 바뀌나요?**



승격단계가 축소되는 것을 감안하여 책임 직급의 초임이 기존 대비 3% 정도 인상되며

연말 성과평가 결과에 따라 적용되는 평가인상률은 직급 구분 없이 기존 대비 0.5% 상향 조정됩니다.



또한 일부 외부 프로젝트 PM 역할을 수행하는 경우에 한하여 PM수당을 지급하며

대상 및 지급 기간은 프로젝트 투입 상황에 따라 수시로 변동 될 수 있습니다.





**📢 고정OT 시간은 어떻게 바뀌나요?**

기존 52시간이던 고정OT 시간이 30시간으로 축소 변경됩니다.

줄어든 22시간에 한하여 실제 근로 한 시간만큼 연장근무수당을 지급합니다.



고정OT시간이 줄어들어도 기존 연봉이 변함 없도록 기본급을 확대하여 임금체계를 설계하였습니다.

따라서 기본급을 기준으로 산정되는 연장근무수당과 연차수당 등은 기존 대비 상승하게 됩니다.





**📢 휴일근무는 어떻게 바뀌나요?**



토요일/일요일 및 휴일 근무 시 사전에 신청/승인된 건에 한하여 출/퇴근 시간을 통해 계획대로 근무 이력이 확인되면 대체휴일이 자동으로 생성됩니다.

반일/하루 단위로 신청이 가능하며 반반차 단위로는 신청을 받지 않습니다.





**📢 시차출퇴근제도는 유지되나요?**



지금과 동일하게 8:00/9:00/10:00 시차출퇴근제도는 동일하게 운영합니다.

다만 외부 프로젝트 수행 시 해당 고객사 근무시간에 맞추어 8:30/9:30 에 출근할 수도 있습니다.

필요시 경영지원팀과 사전 협의 부탁드립니다.





**📢 연장근로시간 산정은 몇시부터 인가요?**



시차출퇴근제도를 운영하고 있으므로 출퇴근 시간이 개인별로 다를 수 있습니다.

따라서 연장근로 기준시간도 다르게 적용되게 됩니다.



아래의 사례를 통해서 연장근로 기준시간에 대해서 설명해 드립니다.



**[CASE] 시차출퇴근제 : 9시**

9:00 - 18:00 정규근로, 12:00 - 13:00 점심휴게(1시간) 포함

18:00 - 18:30 저녁 휴게(30분)

18:30 - 20:00 고정OT 30시간 포함(1.5시간/일)

20:00 이후 부터 실제 근무한 시간 만큼 연장근무시간으로 산정됨



연장근무는 사전에 신청/승인된 건에 한하여 인정되며 출/퇴근 체크를 통해 실제 근무이력이 확인 되어야만 수당이 발생됩니다.





**📢 미사용연차 보상 한도는 어떻게 달라지나요?**



미사용연차 보상한도는 발생연차의 30%로 모든 직원이 동일합니다.

다만 장기간 외부 프로젝트 수행으로 인하여 연차를 적절하게 사용하지 못한 직원의 경우 미사용연차 보상한도를 확대하여 운영할 수 있습니다.



- 대상자 : 1년간 외부 프로젝트를 9개월 이상 수행한 자

- 보상한도 : 30% → 70% 로 확대 운영





**📢 대체휴일과 연차를 적절하게 사용하지 못한 경우 이월이 가능한가요?**



원칙적으로는 지정한 대체휴일, 보상한도 외 연차는 기한 내에 모두 사용하셔야 합니다.

다만  특별한 사유(업무적인 사유 한정)에 의하여 연차를 적절하게 사용하지 못한 경우 소속 팀장님과 상의/경영지원팀과 협의하여 최종 대표이사 승인된 사항에 한하여 이월 해드리고 있습니다.
