회사지정 모델: 개인매각/반납 선택가능 

개인지정모델: 개인매각 필수

퇴사시 반납/매각정책

개인지정모델 필수매각, 회사지정모델 필수반납

디스플레이 : 15.6" FHD IPS, 16:9, Anti Glare (HD 웹캠)
프로세서 : 13세대 인텔 코어 i7-1360P 프로세서(18MB 캐시, 2.2GHz 터보부스트 5.0GHz)

메모리 : 16GB LPDDR4x 4266MHz (온보드, 확장 불가)
SSD : 512GB M.2 NVMe Gen4 (256GB x 2) 
그래픽 : intel Iris Xe Graphics (내장형)

무게 : 1140g

색상: 화이트 



**개발직군용→ Lenovo ThinkBook 16p G5 지급 **

디스플레이 : 16" 3.2K (3200 x 2000), IPS, Anti-Glare, Non-Touch, 100%DCI-P3, 430 nits, 165Hz, Narrow Bezel, Low Blue Light

프로세서 : 14th Generation Intel® Core™ i9-14900HX Processor (E-cores up to 4.10 GHz P-cores up to 5.80 GHz)

메모리 : 32 GB DDR5-5600MHz (SODIMM) - (2 x 16 GB)

SSD : 1 TB SSD M.2 2280 PCIe Gen4 TLC

그래픽 : NVIDIA® GeForce RTX™ 4060 Laptop GPU 8GB GDDR6

무게 : 2340g

색상 : storm gray



**개인지정 모델**
