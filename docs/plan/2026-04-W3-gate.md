- **실효 변경:** DEVELOPER 역할(KNOWLEDGE_UPDATE + SYSTEM_ACCESS_SECRET, **KNOWLEDGE_REVIEW 없음**)이 wiki 검색/AI 결과에서 RESTRICTED 페이지를 더 이상 받지 못함

### [정상] eval fixture expectedPages 큐레이션 예정

- `page-qa.jsonl`의 `expectedPages`는 대부분 `[]` 또는 가상 slug
- 실제 위키 페이지 ingest 후 큐레이션 필요 (Phase-W4 전 ≥ 80% recall 목표)
- `page-first-baseline.ts` shortlist 호출부 TODO placeholder — 실 DB 연결 후 활성화
