## 발견된 이슈 (X7 최종)

### [P1/블로킹] web type-check FAIL

- **파일:** `apps/web/app/(app)/wiki/[workspaceId]/[...path]/__tests__/permission.test.ts:112`
- **에러:** TS2532 "Object is possibly 'undefined'" — `EXPECTED_MATRIX[role][sensitivity]`
- **원인:** `EXPECTED_MATRIX`가 `Record<string, Record<Sensitivity, boolean>>`이므로 `[role]` 접근 결과가 `| undefined`
- **수정:** `EXPECTED_MATRIX[role]!` non-null assertion 또는 `const entry = EXPECTED_MATRIX[role]; expect(entry).toBeDefined(); expect(entry![sensitivity])...`
- **빌더 수정 요청**

### [P2/비블로킹] en.json Wiki 네임스페이스 누락

- **파일:** `apps/web/messages/en.json`에 `"Wiki"` 최상위 키 없음
- **영향:** `forbidden.tsx`가 `getTranslations('Wiki')` → `t('accessDenied')` 호출. ko.json에는 `Wiki.accessDenied: "접근 권한이 없습니다."` 존재
- **리스크:** en locale 사용자에게 fallback 동작에 따라 키 이름이 그대로 노출될 수 있음
- **수정:** en.json에 `"Wiki": { "accessDenied": "Access denied." }` 추가 필요
- **빌더 수정 요청**

### [P3/관찰] admin nav에 review-queue 링크 미등록

- **파일:** `apps/web/app/(app)/admin/layout.tsx` — NAV_ROUTES에 `/admin/wiki/review-queue` 없음
- **영향:** 사이드바에서 직접 접근 불가, URL 직접 입력으로만 진입 가능
- **수정:** NAV_ROUTES에 `{ href: '/admin/wiki/review-queue', key: 'wikiReviewQueue' }` 추가 + ko/en.json `Admin.nav.wikiReviewQueue` 키 추가
- **빌더 수정 요청 (또는 Phase-W4 후속 작업으로 분류)**
