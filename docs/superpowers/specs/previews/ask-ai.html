외주 계약직(contractor) 사원이 위키에서 Infrastructure Runbook을 편집할 수 있는지 알려줘. 필요하면 RBAC 매트릭스 문서도 근거로 붙여줘.

AI
Ask AI 10:42 · 2.1s · policy lane

결론부터: 외주 계약직(contractor)은 편집 불가, 읽기만 가능합니다. 단 해당 문서가 sensitivity: internal 이하일 때에만.

근거:

  • RBAC 매트릭스 §3.2에 따르면 contractor 역할은 wiki:read 권한만 보유, wiki:write·wiki:review는 제외.
  • Infrastructure Runbook 카테고리는 sensitivity: confidential로 분류되어 있어, contractor에게는 목록조차 노출되지 않습니다.
  • 긴급 편집이 필요하면 wiki:emergency-edit 임시 권한을 platform-lead가 4시간 한도로 발급할 수 있습니다 (감사 로그 남음).

요약 표:

역할            wiki:read  wiki:write  wiki:review
employee        ✓          ✓ (own)     ✗
reviewer        ✓          ✓           ✓
platform-lead   ✓          ✓           ✓ (all)
admin           ✓          ✓           ✓ (all)
contractor      ✓ (scope)  ✗           ✗
